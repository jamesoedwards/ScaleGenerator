# ScaleGenerator

Scale generator for music practise. Gives you a key and a scale from this list:
- Pentatonic
- Blues
- Lydian
- Lydian dominant (b7)
- Lydian augmented (#5)
- Ionian
- Harmonic major
- Melodic major
- Mixolydian
- Dorian
- Aeolian
- Melodic minor
- Harmonic minor
- Phrygian
- Altered phrygian
- Locrian
- Super locrian/Altered dominant
- Locrian natural 2?
- Whole tone
- Half diminish
